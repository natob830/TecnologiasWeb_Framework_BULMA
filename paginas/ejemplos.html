Ejemplos de Uso de Buenas Prácticas en Maquetación Web

Ejemplo 1: Estructura Básica de una Página Web

Usar etiquetas semánticas mejora la accesibilidad y el SEO de un sitio web. A continuación se muestra un ejemplo de cómo organizar un contenido básico utilizando HTML5 de forma semántica. 


<header>
    <h1>Bienvenido a Mi Sitio Web</h1>
    <p>Este es un ejemplo de estructura semántica</p>
</header>

<nav>
    <ul>
        <li><a href="inicio.html">Inicio</a></li>
        <li><a href="sobre-nosotros.html">Sobre Nosotros</a></li>
    </ul>
</nav>

<section>
    <h2>Sección de Introducción</h2>
    <p>Aquí puedes hablar sobre tu sitio web o negocio. Esto debería estar en un artículo o sección específica</p>
</section>

<footer>
    <p>© 2025 Mi Sitio Web</p>
</footer>

Ejemplo 2: Uso de Artículos y Asides

El uso de la etiqueta <article> es ideal para contenido independiente como artículos o publicaciones. También puedes usar <aside> para contenido relacionado pero no esencial. 


<article>
    <h2>Artículo de Blog</h2>
    <p>Este es un ejemplo de un artículo de blog que puede ser compartido de forma independiente.</p>
</article>

<aside>
    <h3>Notas del Autor</h3>
    <p>Este es un contenido relacionado con el artículo pero no es esencial para la lectura principal.</p>
</aside>

Ejemplo 3: Lista de Contenidos

Una lista semántica ayuda a organizar el contenido de manera clara y accesible. Aquí un ejemplo de una lista ordenada de pasos para una tarea. 


<ol>
    <li>Paso 1: Investigar</li>
    <li>Paso 2: Planificar</li>
    <li>Paso 3: Ejecutar</li>
    <li>Paso 4: Evaluar</li>
</ol>
